# telegram-botV1
Ini adalah sebuah projek bot telegram yang dibangun menggunakan bahasa pemrograman python secara menyeluruh. Bot ini dijalankan dan di install menggunakan Railway. Jika ingin dijalankan menggunakan terminal maka ubah beberapa kode yang ada di dalamnya atau jika tidak ingin sulit untuk merubahnya nanti akan saya upload V2 yang lebih setabil dan tidak banyak bug di dalamnya, serta gratis di jalankan. Kekurangan jika dijalankan menggunakan terminal bot tidak akan nyala 24/7, jika terminal kamu di tutup maka bot akan berhenti berjalan.
